When using the Debut Social feature there is some initial configuration that must be done prior to using.

Service Links

  • Under Administer/Site configuration, choose Service Links.
  • Under "general settings" select for which content types you would like to add share buttons.
  • The rest of the configuration does not require changes unless you wish different settings.
  • Save your configurations.
  • Next, under the "services" tab, select the providers you wish to have show as buttons.
  • Include "Forward" to enable emailing to a friend.
  • Rearrange to the order you would like.
  • Save your configurations.

CAPTCHA

  • If you are using Forward to enable emailing, then be sure to download and enable CAPTCHA (http://drupal.org/project/captcha).
  • Configure CAPTCHA under Administer/ User management.
  • Check the box "Add CAPTCHA administration links to forms."
  • Next test generate a form of the type you want to add a CAPTCHA to, in this case email. Click the box where it notes no CAPTCHA enabled and do so.

Follow

  • Under Administer/Site building, select "site follow links".
  • Insert the URL for any service you want to show, "follow us on".
  • Save your configurations.
  • If you want to change any of the configuration of "site follow" and "user follow" blocks do so via Administer/Site building/Blocks. For each block, you can select "Configure" and here choose the display, for example "Follow us" vs. "Follow site name".
